# Prafful Gupta Portfolio

A personal portfolio site for Prafful Gupta, a backend-focused software engineer building scalable APIs, distributed backend systems, and performance-heavy products.

## Highlights

- Strong positioning around backend engineering and distributed systems
- Story-driven sections covering background, trajectory, experience, and skills
- Live project links for the collaboration platform and distributed rate limiter
- Downloadable resume bundled with the site
- Built with React and Vite

## Local Development

```bash
npm install
npm run dev
```

The app runs locally on the Vite dev server and builds for production with:

```bash
npm run build
```

## Tech Stack

- React 19
- Vite 8
- Plain CSS
- ESLint
